
### project start
```
- mkdir firebase-chat-app
- npx create-react-app ./
- npm install react-router-dom --save
```

### redux
```
- npm install redux
- npm install react-redux
- npm install redux-promise
- npm install redux-thunk
```

```
- https://firebase.google.com
- npm install firebase
```
